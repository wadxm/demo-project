# Demo Personal Homepage

A polished personal homepage demo built with Vite and React. The project is intentionally small enough for agent experiments while still behaving like a real frontend app with a build, tests, assets, and deployment workflow.

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm test
npm run build
```

## Deployment Target

Production URL: https://demo.xiongdianpku.com

The static build output is `dist/`.
